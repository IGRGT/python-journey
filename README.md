# python-journey

Documented log of me learning Python, from zero, one project at a time. Not a polished portfolio piece — this is the arc: what I built, what broke, what I fixed.

## What's here

- **amiguito.py** — a virtual pet, terminal version. Class-based, saves state to JSON, stats decay over time.
- **app.py** — same pet, but as a Flask web app. Buttons in the browser (feed/play/rest/drink) update its stats live.
- **guess_the_number.py** — number guessing game with difficulty levels, guess limits, and a best-score tracker.

## Why

Studying CS fundamentals for a Master's in Informatics starting Fall 2026. This repo is where I practice, in the open, as I go.
